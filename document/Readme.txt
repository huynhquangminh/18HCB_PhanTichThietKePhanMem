Hướng dẫn cài đặt ứng dụng quản lý thư viện
Bước 1: Vào thư mục Install, chọn QuanLyThuVienSetup.msi hoặc setup.exe để bắt đầu cài đặt
Bước 2: Màn hình Welcome to the QuanLyThuVien Setup Wizard xuất hiện, chọn Next. 
Bước 3: Tại màn hình Select Installation Folder ta tiến hành chọn đường dẫn thư mục cài đặt 
ứng dụng. Sau khi chọn xong tiến hành nhấn Next.
Bước 3: Tại màn hình Confirm Installation ta tiến hành chọn Next để bắt đầu quá trình cài đặt.
Bước 4: Màn hình cài đặt Installing QuanLyThuVien xuất hiện, quá trình cài đặt đang được
tiến hành.
Bước 5: Sau khi cài đặt thành công, màn hình Installation Complete xuất hiện, chọn Close
để hoàn tất quá trình cài đặt.

Lưu ý: Phải cài SQL Server Express trước và chạy script sql trong file Db_final.sql thuộc thư mục
Database để có thể chứa dữ liệu trong quá trình thực hiện chương trình.
